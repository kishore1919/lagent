OpenRouter Key Required

Welcome! Lagent utilizes OpenRouter to run advanced LLM reasoning and coordinate terminal-level tools.

Notice: Your .env file is missing or doesn't have OPENROUTER_API_KEY set.

To run the agent in fully autonomous mode, please create a .env file in your project directory with your API key:

OPENROUTER_API_KEY=your_key_here
OPENROUTER_MODEL=openai/gpt-4o-mini
MCP_PORT=9000

Alternatively, you can test-drive Lagent in Demo / Simulation Mode where we simulate agent thought-loops and system checks!
